O que é?

  • HTML significa Hyper Text Markup Language
  • HTML é a linguagem padrão para a criação de página Web
  • HTML descreve a estrutura de uma página Web
  • HTML consiste uma uma serie de elementos(tags)
  • As tags HTML dizem ao navegador como mostrar o conteúdo
  • As tags representam pedaços de conteúdo como "isso é um título" ou "isso é um paragrafo", etc.
