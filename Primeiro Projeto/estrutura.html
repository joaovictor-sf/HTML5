Estrutura
Uma página html é composta principalmente de tags. Uma tag é composta de caracteres especificos dentro de simbolos de maior e menor.

Exemplo: <htmll>

Para criar uma pagina web é necessario primeiro colocar <!DOCTYPE>, que é o que define o tipo do documento, no caso tipo HTML5, na primeira linha da página e depois colocar todo o código dentro da tag <html>.

Uma página html é divida em duas partes, a cabeça(head) e o corpo(body).

No head é onde fica as informações da página que não aparecem diretamente para o úsuario e no body fica as informações que aparecem ao usuario.
